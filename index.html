Métodos de Strings

  • split(char): Retorna uma lista com os elementos separados pelo char
    a = '1+2+3+4+5+6'
    a.split('+')

  • strip(chars): Retorna uma string onde os chars da direita e da esquerda foram removidos
    a = ' !!! STRING DE RETORNO! !! ! !'
    a.strip(' !')

Métodos de Strings

  • find(substring): Retorna a posição da primeira substring se for encontrada, senão retorna -1
    a = 'um texto qualquer'
    a.find('tex') # 3 a.find('abc') # -1

  • lower(), upper(), title()

  1. A partir da string “!! ! a;b;c;d;e;f;gh!#########” gere o resultado: [“a”, “b”, “c”, “d”, “e”, “f”, “g”]
  2. A partir da string “ring ring! - hello!“ gere o resultado:
    “hello!”
  3. Transformar a string “isso deve ser bom” para “Isso Deve Ser Bom”
  4. Transformar a string “abacate azul” em “4b4c4te 4zul”

Dicionários

  • Um dicionário é uma lista de associações compostas por:
    • uma chave única
    • e estruturas correspondentes
  • O programador pode definir a chave do item
  • Representados por chaves “{ }”

Dicionários

Dicionários

>>> dicionario = {'aluno' : 'Carlos', 'notas': [9, 10], 22: 100}
>>> dicionario['aluno']
'Carlos'
>>> dicionario['notas']
[9, 10]
>>> dicionario[22]
100

Dicionários

  • Inserir item: dicionario[chave] = obj
  • Remover item:
    • dicionario.pop(chave)
      # Se não encontrar a chave retorna erro
    • dicionario.pop(chave, retorno)
      # Se não encontrar a chave retorna o valor de retorno
  • Remover todos itens: dicionario.clear()
  • Verificar se existe item: dicionario.has_key(chave)

Prática

Leia o nome e a nota de 3 alunos e guarde em um dicionário

Módulos

  • Módulos são arquivos fonte que podem ser importados para um programa
  • Podem conter qualquer estrutura do Python
  • Os módulos são carregados através da instrução import

Módulos

  • importação absoluta
    import os
    print os.name
  • importação relativa
    from os import name
    print name

Módulos

  • importação relativa com "*"
    from os import *
    print name

Exceções

  • Quando ocorre uma falha em tempo de execução uma exceção é gerada
  • Se a exceção não for tratada, irá interromper a execução do programa

Exceções

Executar (1 / 0)

Exceções

  • Se não for passada nenhuma exceção todas serão tratadas
  • O tratamento de exceções pode possuir um bloco else, que será executado quando não ocorrer nenhuma exceção
  • O tratamento de exceções pode possuir um bloco finally, que será executado de qualquer forma
  • Novos tipos de exceções podem ser definidos através de herança a partir da classe Exception

Classes

  • Novos objetos são criados a partir das classes através de atribuição
  • O construtor é um método especial, chamado __new__()
  • Após a chamada ao construtor, o método __init__() é chamado para inicializar a nova instância.
  • Não existem variáveis e métodos privados (_*)
